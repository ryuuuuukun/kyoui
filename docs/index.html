コンセプト

「恐怖」で心を動かす

いつもと同じ道、同じ景色、同じ空気。でもそこに「恐怖」が加わるとどうなるだろう。 きっといつもとは違った風に見えるはずです。 「恐怖」には、当たり前の世界を非現実的な世界へと導く力があると思います。 私たちは、その「恐怖」を使って人の心が動く瞬間をつくるクリエイター集団です。
